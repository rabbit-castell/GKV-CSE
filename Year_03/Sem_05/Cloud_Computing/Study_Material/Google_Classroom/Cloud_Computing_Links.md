## 📚 Section 1: Lab Practicals Checklist

### **Part A: Setup & Configuration**
- [ ] **Practical 1:** Write the steps to create an account on IBM Cloud.
  - *Note:* Requires IBM SkillsBuild registration to access feature codes.

### **Part B: Generative AI (LLMs & Text Generation)**
- [ ] **Practical 2:** Write the steps to generate and compare different LLM models using IBM Cloud Pak for Data.
  - *Objective:* Compare outputs between models like Granite, FLAN, and Llama.
- [ ] **Practical 4:** Write the steps for note generation using IBM Watsonx.
  - *Objective:* Use Prompt Lab to summarize or generate text.
- [ ] **Practical 7:** Write the steps to explain how to use RAG (Retrieval-Augmented Generation) in IBM Cloud.
  - *Objective:* connecting LLMs to external documents for factual accuracy.

### **Part C: Natural Language Processing (NLP) & Sentiment**
- [ ] **Practical 3:** Write the steps to do sentiment analysis based on text using IBM Watsonx.
- [ ] **Practical 6:** Write the steps for English to French translation using IBM Watsonx.
- [ ] **Practical 10:** Write the steps for translation from English to Spanish.

### **Part D: Code Generation & Conversational AI**
- [ ] **Practical 5:** Write the steps for building a chatbot using IBM Watsonx.
  - *Tool:* Watsonx Assistant.
- [ ] **Practical 8:** Write the steps to generate code using IBM Watsonx.
- [ ] **Practical 9:** Write the steps for code translation using IBM Watsonx.
  - *Example:* Translating Python code to Java or C++.
